| Command              | Description                      |
|----------------------|----------------------------------|
| `./build.sh`         | Cleans and compiles cheeseDOS    |
| `./build.sh clean`   | Removes all build files          |
| `./build.sh run`     | Run’s cheeseDOS in QEMU          |
| `./build.sh run-kvm` | Run’s cheeseDOS in QEMU with KVM |
